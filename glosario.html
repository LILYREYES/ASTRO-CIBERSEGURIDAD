Phishing

Técnica de engaño para obtener información confidencial como contraseñas o tarjetas de crédito.

Malware

Software malicioso diseñado para dañar o robar información en un sistema.

Firewall

Dispositivo o software que controla el tráfico entre redes para proteger los sistemas.
